GET lock/BoardID  -->  (UserName,LockID,ExpirationTime) or 404
PUT lock/BoardID  -->  (UserName,LockID,ExpirationTime) or failure code   (compiles the board)
DELETE lock/BoardID  -->  success or failure code

GET meta/BoardID  -->   (Lock,BoardOwner,BoardMetaData)

GET board/BoardID  -->   (BoardOwner,BoardMetaData,Board,CompiledBoard) or failure   (if lock is yours, CompiledBoard includes rules + appropriate tools; otherwise it's empty)

PUT move/BoardID (LockID,TimeSinceLock,X,Y,State)  -->  success or failure code   (fails if not your lock, move illegal, etc)

GET type/TypeName  -->  (TypeName,TypeInfo) or 404            (mirrors S3)
PUT type/TypeName  -->  (TypeName,TypeInfo) or failure code   (fails if no designer privileges, not your namespace, etc)

GET status  -->  (UserName,Privileges,BoardsOwned,CashLevel,PixelInventory)

GET price/Type  -->  (PricePerUnit)

PUT buy/Type  (PixelsToBuy)  -->  (PixelInventory) or failure code
